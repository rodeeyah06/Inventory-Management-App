# Inventory Management Application

## Overview
This project is an **Inventory Management Application** designed to manage products, track stock levels, and handle basic inventory operations efficiently.

The application focuses on applying real-world business logic to software, simulating how inventory systems work in retail and enterprise environments.

---

## Features
- Add new products to inventory  
- Update product details  
- Track stock quantity  
- Remove products from inventory  
- Prevent invalid operations (e.g. negative stock)
